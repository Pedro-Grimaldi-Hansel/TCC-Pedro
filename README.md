# Sistema de Recomendação – TCC

Este repositório contém os experimentos e implementações
relacionados ao TCC sobre sistemas de recomendação baseados
em similaridade e aprendizado de máquina.

## Estrutura do Projeto
- `data/`: conjuntos de dados
- `notebooks/`: análises exploratórias
- `src/`: código-fonte dos algoritmos

## Tecnologias
- Python 3
- NumPy
- Pandas
- scikit-learn
